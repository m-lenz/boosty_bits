# Boosty Bits

Basic setup for BOOST/C++ tests. 
 example for:
 * functions
 * suits
 * fixtures 
 * assertions
 
 ## supported operations are
 1. make       (build files)
 2. make test  (run test and generate XML)
 3. make clean (remove built files)

## needed dependencies are:
* boost-lib
* make / cmake
* jenkins (target-plattform for test)
